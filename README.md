# Morphowebshop — Интернет-магазин электроники

Современный интернет-магазин электроники с полным функционалом для покупки товаров, управления корзиной, оформления заказов и отслеживания доставки.

## Технологии

- **Next.js 16** (App Router) — фреймворк React для серверного рендеринга
- **TypeScript** — строгая типизация
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **Shadcn/ui** — компоненты пользовательского интерфейса
- **Zustand** — управление состоянием приложения
- **Sonner** — уведомления

## Дизайн

Сайт выполнен в зеленой цветовой гамме с чистым и современным интерфейсом:
- Основной цвет: зеленый (HSL: 146 54% 59%)
- Дополнительные цвета: светло-зеленый, темно-зеленый
- Шрифт: Inter (с поддержкой кириллицы)
- Адаптивный дизайн для всех устройств

## Реализованный функционал

### 1. Главная страница (/)

**Функции:**
- Слайдер с промо-баннерами (автоматическая смена каждые 5 секунд)
- Фильтрация товаров по категориям (Все, Смартфоны, Ноутбуки, Наушники, Видеокарты)
- Поиск товаров по названию
- Сортировка: по популярности, цене (возрастание/убывание)
- Сетка товаров с карточками
- Пагинация (12 товаров на странице)

**Как использовать:**
1. Откройте главную страницу
2. Используйте поле поиска в шапке для поиска товаров
3. Выберите категорию в фильтрах слева
4. Выберите сортировку из выпадающего списка
5. Нажмите на карточку товара для просмотра деталей
6. Нажмите "В корзину" для добавления товара

### 2. Страница товара (/product/[id])

**Функции:**
- Крупное изображение товара
- Название, рейтинг, количество отзывов
- Детальное описание товара
- Список характеристик (процессор, память, экран и т.д.)
- Цена с выделением
- Выбор количества товара
- Добавление в корзину
- Индикатор наличия товара на складе

**Как использовать:**
1. Перейдите на страницу товара из каталога
2. Ознакомьтесь с описанием и характеристиками
3. Выберите нужное количество (используйте кнопки +/-)
4. Нажмите "Добавить в корзину"
5. Используйте кнопку "Купить сейчас" для быстрого перехода к оформлению

### 3. Аутентификация (/auth)

**Функции:**
- Переключение между вкладками "Вход" и "Регистрация"
- Форма входа (email, пароль)
- Форма регистрации (имя, email, телефон, пароль, подтверждение пароля)
- Валидация полей
- Автоматическое перенаправление после входа

**Как использовать:**

**Вход:**
1. Перейдите на страницу /auth
2. Введите email и пароль
3. Нажмите "Войти"
4. После успешного входа вы будете перенаправлены на главную

**Регистрация:**
1. Переключитесь на вкладку "Регистрация"
2. Заполните все поля формы
3. Убедитесь, что пароли совпадают
4. Нажмите "Зарегистрироваться"
5. После регистрации вы будете автоматически авторизованы

**Тестовый аккаунт:**
- Email: test@example.com
- Пароль: любой

### 4. Личный кабинет (/profile)

**Функции:**
- Четыре вкладки: Профиль, Заказы, Адреса, Безопасность
- Редактирование личных данных (имя, email, телефон)
- Просмотр истории заказов с деталями
- Фильтрация заказов по статусу
- Управление адресами доставки (добавление, редактирование, удаление)
- Установка адреса по умолчанию
- Изменение пароля
- Выход из аккаунта

**Как использовать:**

**Вкладка "Профиль":**
1. Войдите в систему
2. Нажмите на иконку профиля в шапке
3. Отредактируйте данные
4. Нажмите "Сохранить изменения"

**Вкладка "Заказы":**
1. Переключитесь на вкладку "Заказы"
2. Используйте фильтры для поиска нужных заказов
3. Нажмите "Подробнее" для просмотра деталей заказа
4. Нажмите "Отследить" для отслеживания доставки

**Вкладка "Адреса":**
1. Переключитесь на вкладку "Адреса"
2. Нажмите "Добавить адрес"
3. Заполните форму (название, улица, город, индекс, страна)
4. Нажмите "Сохранить"
5. Используйте "Изменить" для редактирования
6. Используйте "Сделать основным" для установки адреса по умолчанию

**Вкладка "Безопасность":**
1. Переключитесь на вкладку "Безопасность"
2. Введите текущий пароль
3. Введите новый пароль дважды
4. Нажмите "Изменить пароль"

### 5. Корзина (/cart)

**Функции:**
- Список всех товаров в корзине
- Изображение, название, цена каждого товара
- Изменение количества товара (кнопки +/-)
- Удаление товара из корзины
- Подсчет общей суммы
- Количество товаров в корзине
- Переход к оформлению заказа
- Сообщение о пустой корзине

**Как использовать:**
1. Добавьте товары в корзину на страницах товаров
2. Нажмите на иконку корзины в шапке или перейдите на /cart
3. Измените количество товаров при необходимости
4. Удалите ненужные товары кнопкой с иконкой корзины
5. Нажмите "Оформить заказ" для перехода к оплате

### 6. Оформление заказа (/checkout)

**Функции:**
- Трехшаговый процесс оформления:
  - Шаг 1: Контактная информация (имя, email, телефон)
  - Шаг 2: Адрес доставки (улица, город, индекс, страна)
  - Шаг 3: Способ оплаты (карта, наличные, перевод)
- Выбор существующего адреса из профиля
- Предзаполнение данных для авторизованных пользователей
- Сводка заказа с товарами
- Расчет стоимости (товары, доставка, итого)
- Валидация всех полей перед оформлением
- Генерация трек-номера для отслеживания

**Как использовать:**
1. Перейдите на страницу оформления заказа из корзины
2. **Шаг 1:** Заполните контактные данные и нажмите "Продолжить"
3. **Шаг 2:** Выберите адрес из сохраненных или введите новый, нажмите "Продолжить"
4. **Шаг 3:** Выберите способ оплаты:
   - Банковская карта: введите данные карты
   - Наличные при получении: подтвердите выбор
   - Банковский перевод: получите реквизиты
5. Нажмите "Оформить заказ"
6. После оформления вы получите трек-номер для отслеживания

### 7. Отслеживание заказа (/tracking/[trackingNumber])

**Функции:**
- Визуализация статуса доставки
- Timeline с этапами доставки:
  - Заказ оформлен
  - Заказ собирается
  - В пути
  - Доставлен
- Прогресс-бар доставки
- Информация о заказе (номер, дата, статус)
- Список товаров в заказе
- Адрес доставки
- Детализация стоимости
- Контактная информация

**Как использовать:**
1. Получите трек-номер после оформления заказа
2. Перейдите на /tracking/[ваш-трек-номер] или
3. Нажмите "Отследить" в истории заказов в профиле
4. Просмотрите текущий статус доставки
5. Проверьте детали заказа и адрес доставки

### 8. Шапка сайта (Header)

**Функции:**
- Логотип с ссылкой на главную
- Поле поиска товаров
- Навигационное меню (Каталог, Акции, О нас, Контакты)
- Иконка профиля (для авторизованных пользователей)
- Иконка корзины с счетчиком товаров
- Адаптивное меню для мобильных устройств

### 9. Подвал сайта (Footer)

**Функции:**
- Информация о компании
- Ссылки на разделы (О нас, Доставка, Оплата, Контакты)
- Политика конфиденциальности и условия использования
- Контактная информация (телефон, email, адрес)
- Копирайт

## Управление состоянием

Приложение использует три основных хранилища:

### Auth Store (authStore)
- Управление пользователем (вход, регистрация, выход)
- Хранение данных пользователя
- Проверка авторизации

### Cart Store (cartStore)
- Добавление/удаление товаров
- Изменение количества
- Подсчет общей суммы
- Сохранение в localStorage

### Order Store (orderStore)
- Создание заказов
- История заказов
- Управление адресами доставки
- Отслеживание статуса

---

## Особенности реализации

1. **Полностью клиентское приложение** — все данные хранятся в localStorage браузера
2. **Реактивное состояние** — изменения отображаются мгновенно на всех страницах
3. **Адаптивный дизайн** — корректно работает на всех устройствах
4. **Валидация форм** — все поля проверяются перед отправкой
5. **Уведомления** — всплывающие сообщения о действиях пользователя
6. **Персистентность** — данные сохраняются между сеансами
7. **SSR-совместимость** — правильная работа с Next.js App Router

## Примеры использования

### Пример 1: Покупка товара

1. Зайдите на главную страницу
2. Найдите товар через поиск или фильтры
3. Нажмите на карточку товара
4. Выберите количество и нажмите "В корзину"
5. Перейдите в корзину
6. Нажмите "Оформить заказ"
7. Заполните контактные данные
8. Укажите адрес доставки
9. Выберите способ оплаты
10. Оформите заказ и сохраните трек-номер

### Пример 2: Управление профилем

1. Зарегистрируйтесь или войдите в систему
2. Нажмите на иконку профиля в шапке
3. Отредактируйте свои данные во вкладке "Профиль"
4. Добавьте адреса доставки во вкладке "Адреса"
5. Просмотрите историю заказов во вкладке "Заказы"
6. Измените пароль во вкладке "Безопасность"

### Пример 3: Отслеживание заказа

1. После оформления заказа скопируйте трек-номер
2. Перейдите по ссылке /tracking/[ваш-трек-номер]
3. Просмотрите текущий статус доставки
4. Или найдите заказ в личном кабинете и нажмите "Отследить"
